전체 설명

사용통계의 제공(감시)

  • 스마트폰에 설치한 앱과 접속량, 접속한 웹사이트의 목록과 횟수 확인과 관리를 보다 쉽고 편하게!!
  • 자녀의 스마트폰 사용 시간량 확인으로 관리의 편의성을 도모합니다.

스마트폰 사용 제한 설정!!

  • 특정 앱, 웹 사이트 사용 제한(게임, 음란물 실행) 직접 차단 설정할 수 있습니다.
  • 사용 시간에 제한을 설정하여 스마트폰 이용시간을 관리할 수 있습니다.

자녀의 위치 확인

  • 자녀의 실시간 위치와 이동경로를 제공합니다.

알리미 기능

  • 항목별로 분류된 지난 메시지를 통하여 쉽게 재확인 가능
    • 위급상황 알림 기능
    • 자녀의 위급상황시 알리는 기능이며 자녀의 위치, 자녀와의 거리, 요청시간을 제공받을 수 있습니다.
    • 긴급요청시 20분간 1분 간격으로 자녀 위치 정보를 제공합니다.
    • 메시지에 자녀와의 즉시 통화 기능을 이용하여 신속히 대응할 수 있습니다.
    • 자녀의 현장상황 청취를 통화여 상황을 파악할 수 있습니다.
    • 안심지역 지정 위치별 알림 기능
    • 설정한 안심지역에서의 자녀 이동상황을 실시간으로 보고받을 수 있습니다.
    • 유해어 알림 기능
    • 설정되어진 유해 단어에 대한 이용시 보고받을 수 있습니다.
    • 문자, SNS, 메신저 등에서 유해단어 발견 시 보호자에 통보합니다.(유해단어 : 폭력, 자살, 왕따, 성추행 등)

상세 설명

앱 메뉴 이용방법

  • 자녀의 오늘, 어제, 일주일 사용량 확인
    • 상단 버튼 기능
    • 최근실행순, 실행횟수순, 최근설치순으로 선택 확인 가능
    • 새로고침버튼 : 새로고침 기능 활성화
    • 이전버튼 : 이전 페이지로 이동
    • 허용/차단 버튼 : 앱의 허용, 차단 실행

사이트 메뉴 이용방법

  • 자녀의 오늘, 어제, 일주일 사용량 확인
    • 상단 버튼 기능
    • 최근접속순, 접속횟수순, 차단횟수순으로 선택 확인 가능
    • 새로고침버튼 : 새로고침 기능 활성화
    • 이전버튼 : 이전 페이지로 이동
    • 허용/차단 버튼 : 사이트의 허용, 차단 실행

시간 메뉴 이용방법

  • 자녀의 오늘, 어제, 일주일 사용량 확인
    • 상단 버튼 기능
    • 시간관리 화면으로 이동
    • on/off버튼 : 설정 기능을 활성화
    • 휴지통버튼 : 설정된 관리 항목을 삭제
    • 이전버튼 : 이전 페이지로 이동
    • 시간관리 항목 설정하기 방법
    • 상단 버튼 클릭
    • 화면내 상단 우측 버튼 클릭
    • 시간 차단 이름 입력
    • 시간설정하기(오전, 오후, 반복요일설정)
    • 상단 버튼 클릭

위치 메뉴 이용방법

  • 자녀의 오늘, 어제, 일주일 사용량 확인 (주소, 시간 제공, 해당 항목 클릭시 지도 확인 가능)
    • 상단 버튼 기능
    • 안심지역관리 화면으로 이동
    • on/off버튼 : 설정 기능을 활성
    • 휴지통버튼 : 설정된 관리 항목을 삭제
    • 이전버튼 : 이전 페이지로 이동
    • 안심지역관리 항목 설정 방법
    • 상단 버튼 클릭
    • 상단 우측 버튼 클릭
    • 안심지역명 입력
    • 지역명 입력 : 지역명 입력 후 돋보기 클릭
    • 적용 지역 반경 선택
    • 우측 상단 클릭
    • 안심지역관리 항목 수정 방법
    • 해당항목클릭 : 해당 안심지역관리 항목 수정

알리미 메뉴 이용방법

  • 긴급알림 : 긴급메세지, 기기루팅(탈옥), 유심변경 알림 메시지
  • 알림지역 : 안심지역의 도착과 이탈 알림 메시지
  • 유해어 : 문자, SNS, 메신저 등에서 유해단어 발견 시 보호자에게 통보 (유해단어 : 폭력, 자살, 왕따, 성추행 등)

자녀선택 방법

  • 상단 버튼 클릭하여 등록되어진 자녀의 목록에서 선택

정보수정(자녀관리, 개인정보수정 기능) 방법

  • 상단 버튼 클릭시 하단 항목 선택
    • 자녀관리
    • 정보수정 해당 자녀의 닉네임 변경 가능
    • 자녀 항목의 수정 또는 삭제 기능 실행 가능
    • 개인정보수정
    • 비밀번호 변경 기능
      현재비밀번호, 변경하고자 하는 비밀번호를 입력 후 저장 버튼 클릭
    • 부모정보 변경 기능
      이름, 성별, 전화번호(현 기기의 번호로 지정시 항목 체크 클릭),이메일, 생년월일 변경 가능

로그아웃 방법

  • 상단 "V" 버튼 클릭시 하단 항목 선택
